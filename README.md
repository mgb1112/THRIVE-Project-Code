Practice of Statistical Consulting Final Project: THRIVE Pilot Study

Team Members: Amia Graye, Grace Brooks, Moran Guo

This repository consists of all code used in the statistical analyses for the THRIVE Pilot Study, as it relates to this course.

There were three main aims:
  1. Update Table 1 with medication information
  2. Verify results with LMM Models and medication use
  3. Investigate the missingness within the data set

The data of the analyses is stored in `thrive_data.rds` file.
